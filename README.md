# 2026_01_18_upm_nes_udp_multiplayer

For the fun, let's make a small tool to make int multiplayer UDP game with the nes tool we created.

<img width="1275" height="662" alt="image" src="https://github.com/user-attachments/assets/75ce4581-82e2-4aa1-a522-fa979ff0dbf9" />

For this you need two tool;
- https://github.com/EloiStree/2020_11_29_upm_udp_thread_in_out_gate
  - To listen with a thread at UDP comming to the game.
- https://github.com/EloiStree/2026_01_18_upm_nes_controller_udp
  - To Create your own NES controller
- https://github.com/EloiStree/2024_08_29_ScratchToWarcraft/
  - Convention used.

In the NES workshop we used:
```
Menu Left   1309 2309
Menu Right  1308 2308
Up Arrow    1311 2311  
Down Arrow  1315 2315
Left Arrow  1317 2317
Right Arrow 1313 2313
A Button    1300 2300
B Button    1302 2302
```
